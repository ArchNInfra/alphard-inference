# Alnitak – Inference Service

Alnitak provides a minimal, containerized inference API that can be deployed
to either AWS ECS or GCP Cloud Run.

## Scope
- REST inference endpoint
- Containerized model runtime
- Deployment configs for ECS and Cloud Run
- Basic build and CI setup

## Stack
- Python (FastAPI / Flask)
- Docker
- AWS ECS
- GCP Cloud Run
